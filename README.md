# tarkov-korea-wiki-back

DB 조회 모델 상위 클래스 생성 하고 하위 클래스 만들기 

=> 함수에 클래스랑 쿼리를 매개변수로 보내서 dict를 만들어서 줄 건데, 이 때 사용할 것 

pip3 install 'fastapi[all]'

pip3 install sqlalchemy

pip3 install python-dotenv

uvicorn main:app --reload
